What is csvg? #

Scalable Vector Graphics, SVG is excellent way to draw beautiful images on web. Generally, we use svg editors to make svg images. However, most svg editors do not yet support animation, and they are not flexible for developers. CSVG is a preprocessor with some useful functions for drawing svg and make CSS animation or SMIL animation easier.
